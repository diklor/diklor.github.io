How to my get system bitness?

▲

Linux


Open the CLI to a Linux server by SSH using Putty.
Type the following command line: uname –a

Sample output

$ getconf LONG_BIT

x86_64 GNU/Linux indicates it's a 64 bit Linux kernel.
i386/i486/i586/i686 indicates it's a 32 bit Linux kernel
